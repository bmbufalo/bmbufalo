Last week, we decided that it wasn&rsquo;t enough to just wonder which web browser was better; we had to settle the matter in a battle for the ages.  So we <a href="http://mashable.com/2009/09/21/web-browser-faceoff/">pitted Mozilla Firefox against Google Chrome</a>, gave them some ammunition, and left it up to you to decide <a href="http://mashable.com/2009/09/25/poll-firefox-smacks-down-google-chrome-in-reader-vote/">who was the victor</a>.  In the end, Firefox beat up on Chrome, 4600 votes to 3310.

Everyone had so much fun with the web browser faceoff though, we decided that we had to do it again.  So it&rsquo;s time to take two of the web&rsquo;s most popular apps, put them in the ring, and see which one is the people&rsquo;s choice.  Our competitors this week? This isn&rsquo;t the first time we&rsquo;ve put these two web services in opposite corners: we compared the two <a href="http://mashable.com/2009/06/29/posterous-vs-tumblr/">in a head-to-head</a> comparison (Posterous won, but not by much).  Since then though, Posterous has <a href="http://mashable.com/2009/09/17/posterous-themes/">added custom themes</a> and Tumblr <a href="http://mashable.com/2009/08/05/tumblr-takeover/">broke 255 million pageviews</a>.

So which simple blogging platform deserve to earn the title of winner of this week&rsquo;s <a href="http://mashable.com/tag/web-faceoff">Web Faceoff</a>?  The answer to that question is up to you.
