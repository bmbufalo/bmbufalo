The Jaunty Jackalope is on the loose, bringing a Linux desktop with <a href="http://lifehacker.com/5116597/ubuntu-distributor-wants-to-overhaul-linux-desktop-notifications">streamlined notifications</a>, a <a href="http://lifehacker.com/5130227/speedier-ext4-filesystem-available-in-ubuntu-904">speedier file system</a>, and <a href="http://lifehacker.com/5180833/first-look-at-ubuntu-904-jaunty-jackalope-beta">other upgrades</a> to the free desktop. Grab a <a href="http://lifehacker.com/tag/live-cd/" title="Click here to read more posts tagged LIVE CD">live CD</a> and try it out for yourself.</p> As happens every six months or so, the official Ubuntu servers are swamped with upgraders and downloaders this morning, so you'd be quite the polite person if you <a href="http://lifehacker.com/384325/speed-up-ubuntu-updates-with-a-mirror-server">upgraded from a mirror server</a> or grabbed any of the Ubuntu 9.04 ISO images <a href="http://www.ubuntu.com/getubuntu/downloadmirrors#bt">from an official BitTorrent stream</a>. Thinking about adding Ubuntu to your boot options, but don't want to mess up XP and/or Vista? Check out Adam's guide to <a href="http://lifehacker.com/software/top/hack-attack-how-to-tripleboot-windows-xp-vista-and-ubuntu-193474.php">triple-booting all three of those</a>. You can get even more risk-free by <a href="http://lifehacker.com/5204434/the-beginners-guide-to-creating-virtual-machines-with-virtualbox">running Ubuntu 9.04 in a virtual machine</a>.</p> <p>Ubuntu is, as always, completely free, runs on most any x86-based system.
